## Использованные API

### Qwen API
API от Alibaba Cloud для доступа к языковой модели Qwen. Используется для генерации текстовых ответов на сообщения пользователя в рамках диалога с ботом. Также используется для генерации изображений и видео.
- Документация: https://www.alibabacloud.com/help/en/model-studio
 
### Gemini API
API от Google DeepMind для работы с мультимодальной моделью Gemini. Позволяет обрабатывать текстовые запросы и генерировать интеллектуальные ответы.
- Документация: https://ai.google.dev/docs
 
### DeepSeek API
API от DeepSeek для доступа к одноимённой языковой модели. Применяется для генерации ответов и выполнения задач на понимание текста.
- Документация: https://platform.deepseek.com/docs
 
### CheapShark API
Бесплатный публичный API для получения актуальных скидок и цен на компьютерные игры в различных магазинах (Steam, GOG, Epic Games и др.). Используется в боте для поиска и отображения выгодных предложений на игры.
- Документация: https://apidocs.cheapshark.com

## Трансформер и решаемая задача

| Параметр | Значение |
|---|---|
| **Название модели** | `bert-base-multilingual-cased` |
| **Тип задачи** | `Анализ тональности` |
| **Описание** | `Модель решает задачу классификации текста: определяет тональность (нейтральная / угроза / оскорбление / нерпистойность) входящего сообщения пользователя и на основе этого формирует подходящий ответ бота.` |

## Использование системы контроля версий

Был использован подход разработки GitHubFlow:
1. Основная ветка main
2. Ветка ml_service - разработки микросервиса с нейросетью для определния тональности сообщений
3. Ветка bot - основной сервис телеграм бота

## Запуск проекта

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/VetUps/TelegramBotAI.git
   cd TelegramBotAI
   ```

2. Создайте файл `.env` на основе `.env.example`.

3. Запустите через Docker Compose:
   ```bash
   docker-compose up --build
   ```
